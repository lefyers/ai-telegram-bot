# 🎨 Telegram AI-ассистент — Центр Красок #1

Бот отвечает на вопросы о компании centr-krasok.kz на основе собранной базы знаний.
Использует **Google Gemini 2.0 Flash** (бесплатный tier, 1M токенов/месяц).

---

## Стек

| Компонент | Инструмент |
|-----------|-----------|
| Язык | Python 3.11+ |
| Telegram | python-telegram-bot 21 |
| AI модель | Google Gemini 2.0 Flash |
| Парсинг | requests + BeautifulSoup4 |
| Хостинг | Railway / Render (бесплатно) |

---

### 2. Установить зависимости

```bash
pip install -r requirements.txt
```

---

### 3. Задать переменные окружения

Или создай файл `.env` (не добавляй в git!):
```
TELEGRAM_TOKEN=твой_токен
GEMINI_API_KEY=твой_ключ
```

---

### 4. Запустить бота

```bash
python bot.py
```

---

## Обновление базы знаний

Если сайт компании обновился:
```bash
python scraper.py
```
Откроется файл `company_info_raw.txt` — проверь содержимое и замени `company_info.txt`.
